## Accessibility Features

- **High Contrast Mode**: For visibility in bright sunlight
- **Large Text Mode**: For visually impaired users
- **Screen Reader Support**: Full ARIA labels
- **Keyboard Navigation**: All functions accessible via keyboard
- **Voice Control**: "Hey RoamEN, send message to Pharmacy"
- **Haptic Feedback**: Vibration confirms button presses

## Implementation Technologies

**Frontend**:
- HTML5 + CSS3
- JavaScript (ES6+)
- React or Vue.js (component-based)
- WebSocket for real-time updates
- Progressive Web App (PWA) for offline support

**Backend**:
- Python (aiohttp or FastAPI)
- WebSocket server
- Integration with node firmware
- SQLite for message history

**Deployment**:
- Runs on node's Raspberry Pi
- Accessible via WiFi or Bluetooth
- Can be hosted on central server for infrastructure nodes
